# Polybar Install:
* dependancies:
    * xcb-proto
    * xcb-util-devel
    * xcb-util-wm-devel
    * xcb-util-image-devel
    * cairo-devel
    * python2
    * cmake
* make
    * -Download and unpack latest release tar
    * -move into package directory
    * mkdir build
    * cd build
    * cmake ..
    * make -j$(nproc)
    * sudo make install
